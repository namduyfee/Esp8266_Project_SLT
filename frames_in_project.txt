
TCP:

* delete, open, close:
- 'TCP' + CMD (D O C)
-> trả về 'TCP' + CMD (TCP_ACK | TCP_NACK)


* read
- 'TCP' + CMD (READ) + offset + tot size + max segment size
-> trả về 'TCP' + CMD (TCP_ACK | TCP_NACK)

* return read 
'TCP' + CMD (RETURN READ) + offset + data
...

* end return read 
- 'TCP' + CMD (END RETURN READ) + checksum (của tất cả data)
-> trả về 'TCP' + CMD (TCP_ACK | TCP_NACK)

* start write
- 'TCP' + CMD (start write) + offset_start + tot size
-> trả về TCP + CMD (TCP_ACK | TCP_NACK) + max segment size

* wirte
- 'TCP' + CMD (WRITE) + offset + data
...

* end write
- 'TCP' + CMD (END WRITE) + checksum (của tất cả data)
-> trả về 'TCP' + CMD (TCP_ACK | TCP_NACK)
