
TCP:

* delete, open, close:
- 'TCP' + CMD (D O C).
-> trả về 'TCP' + CMD (TCP_ACK | TCP_NACK)

* start write
- 'TCP' + CMD (TCP_ST_WRITE) + offset_start + tot size
-> trả về TCP + CMD (TCP_ACK | TCP_NACK) + max segment size.

* wirte
- 'TCP' + CMD (WRITE) + offset + data
...

* end write
- 'TCP' + CMD (END WRITE) + checksum (của tất cả data vừa gửi cho node)
-> trả về 'TCP' + CMD (TCP_ACK | TCP_NACK)


ESPNOW:

* open close delete
sender retry + timeout: NOW + CMD + checksum

receiver retry + timeout -> NOW + CMD (NOW_ACK | NOW_NACK) + CMD(O C D request) + checksum

* starte WRITE
sender retry + timeout : NOW + CMD (NOW_ST_WRITE) + offset start + tot size + tot packet + checksum 
-> NOW + (NOW_ACK | NOW_NACK) + CMD (start write) + checksum

* wrtie
NOW + CMD (WRITE) + sequence number + offset + data + checksum
...

* end write
NOW + CMD (END WRITE) + checksum (của tất cả data vừa gửi cho node) + checksum (của segment này)
-> NOW + (NOW_ACK | NOW_NACK) + CMD (end write) + checksum
